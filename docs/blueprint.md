# **App Name**: SwiftChat

## Core Features:

- Chat Interface: Intuitive UI for seamless text-based conversations with typing indicators and timestamps.
- AI-Powered Responses: Leverage rule-based and AI-driven logic to provide context-aware replies, using a tool to select data from a predefined knowledge base for higher-quality answers and improved relevance.
- User Authentication: Secure user login/signup (Email/Phone/Google) with guest chat option and stored chat history for logged-in users, all integrated with Firebase Authentication.
- Admin Panel: Dashboard to view chat logs, manage predefined questions & answers, monitor chatbot performance, and handle user moderation.
- Data Storage: Firestore database for storing chat logs, user data, and predefined responses.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to inspire trust and intelligence.
- Background color: Very light blue (#E8EAF6) to create a calm, clean backdrop.
- Accent color: Soft violet (#9575CD) for subtle highlighting of key UI elements.
- Body and headline font: 'Inter' for a modern and neutral feel. A sans-serif font ensures readability and a contemporary design.
- Use consistent, clear icons from a set like Material Design Icons to maintain visual clarity and usability.
- Employ subtle animations for message delivery and UI transitions to enhance user experience.
